Project Overview

SustainLink is a full-stack B2B web platform designed to connect manufacturers with verified sustainable suppliers. The platform addresses the issue of greenwashing by introducing an expert-based verification system for sustainability claims. Suppliers can submit their claims, which are reviewed and validated by experts before being made visible to manufacturers. The system also includes a trust score mechanism, real-time updates, and role-based dashboards to ensure transparency, credibility, and efficient decision-making in supply chains.ReadMe file (including setup instructions, technologies used, and project overview
